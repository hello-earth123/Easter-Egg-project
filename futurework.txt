1. 스탯창이랑 인벤토리 창 상단 바 드래그하면 움직일 수 있게 수정
2. 스킬창 추가해서 스킬 드래그앤 드롭 하여 사용할 수 있도록
	a. 획득 하지 못한 스킬들은 사용할 수 없으므로 비활성화 되어 있므
3. 타격과 피격시에 데미지 숫자가 화면에 보이기
	a. 유저가 피격받는 데미지와 타격하는 데미지를 구별한다.
4. F 키를 누르면, 보물 상자, 맵 이동 상호작용(scene 전환) (맵 이동 시에는 유저의 모든 정보들이 같이 넘어간다.)
5. 우측 상단에 투명한 퀘스트 창이 항상 떠있다.
6. 아무 창이 없는 경우에 esc 키를 누르면 메뉴 창이 뜬다.
	a. 메뉴 창이 있는 상태에서 esc 키를 누르면 다시 게임 화면으로 돌아간다.
	b. 메뉴 창에는 저장, 음량 조절 칸이 존재한다.
		i) 저장 버튼을 누르면 지금까지의 모든 데이터가 저장된다. -> 로그인 후에 게임을 시작하면 저장된 데이터를 불러와 게			임을 이어간다.
		ii) 음량 조절 버튼을 누르면 게임 내에 음량을 조절할 수 있다.
7. 보스 페이즈 나누기, 보스 크기 키우기
